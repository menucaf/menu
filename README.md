# 🍽️ AI-Powered Restaurant Menu System

A premium, production-ready restaurant menu system with AI chat assistant, built with Cloudflare Workers, and Hugging Face AI.

## ✨ Features

- **AI Menu Assistant** - Chat with our intelligent bot about menu items, recommendations, and more
- **Smart Menu Grid** - Beautiful, responsive menu display with search and filters
- **Location Detection** - Automatically finds nearest restaurant branch
- **Mood Matcher** - Get recommendations based on how you're feeling
- **Secret Menu** - Hidden items that AI can reveal
- **Fun Food Facts** - Learn interesting stories about your food
- **Admin Panel** - Easy menu management interface
